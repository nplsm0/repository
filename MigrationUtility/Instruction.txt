MigrationUtility - инструкция.
1. Подготовка
1.1 Необходимо скомпилировать проект.

2. Настройка
2.1 Указать строку подключения к БД в файле «MigrationUtility.dll.config» в файлах сборки.
2.2 БД на сервере создается автоматически.

3. Использование
3.1 Приложение запускается с помощью командной строки.
3.2 Для запуска используются следующие команды:
  - load - отображает иерархическую структуру подразделений;
  - load -i «id» - отображает иерархическую структуру подразделения с идентификатором id;
  - upload -m jobtitle -p "«path to tsv-file»" - загружает данные должностей в БД;
  - upload -m department -p "«path to tsv-file»" - загружает данные подразделений в БД;
  - upload -m employee -p "«path to tsv-file»" - загружает данные сотрудников в БД.
3.3 Пример bat-скрипта:
  - @echo off
    set path=%cd%
    %path%\MigrationUtility.exe upload -m department -p "C:\path\to\departments.tsv"

4. Рекомендации по загрузке в БД:
4.1 Поскольку некоторые данные зависят друг от друга, рекомендуется загружать их в следующем порядке:
  - jobtitle;
  - department;
  - employee;
  - department.
